# Set an Observable Type as Case-Insensitive

<!-- md:version 5.7 --> <!-- md:permission `[admin] manageObservableTemplate` -->

Make an observable type case-insensitive to ensure that values are treated as identical regardless of letter casing.

When this option is enabled, new observables of this type are automatically normalized to lower case upon creation. Existing observables aren't affected.

<h2>Procedure</h2>

1. {% include-markdown "includes/entities-management-view-go-to.md" %}

2. {% include-markdown "includes/observable-types-tab-go-to.md" %}

3. Select :fontawesome-solid-ellipsis: next to the observable type you want to update.

4. Select **Edit**.

5. Turn on the **Case-insensitive** toggle.

6. Select **Confirm observable type edition**.
