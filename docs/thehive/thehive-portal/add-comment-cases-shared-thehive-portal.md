# Post a Comment to a Case from TheHive Portal

<!-- md:permission `manageComment` -->

[Comments](glossary-thehive-portal.md#comments) allow you to communicate with the Security team about the incident, ask questions, and provide requested information.

<h2>Procedure</h2>

1. [Find and open the case](find-a-case-thehive-portal.md) where you want to add a comment.

    {% include-markdown "includes/filters.md" %}

2. In the case, select :fontawesome-solid-arrow-left:.

    ![Open comment drawer TheHive Portal](/thehive/images/user-guides/thehive-portal/open-comment-drawer-thehive-portal.png)

3. Enter your comment in the text field.

4. Select **Send** to post your comment.

Your comment is visible to all users with access to the case.
