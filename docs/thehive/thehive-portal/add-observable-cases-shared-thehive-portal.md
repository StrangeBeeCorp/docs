# Add an Observable to a Case from TheHive Portal

<!-- md:permission `manageObservable` -->

[Observables](glossary-thehive-portal.md#observables) are specific pieces of evidence related to the security incident, such as suspicious email addresses, file names, or website links that help the Security team investigate the case.

If you need to share whole document or files, see [Upload an Attachment to a Case from TheHive Portal](add-attachment-cases-shared-thehive-portal.md).

!!! note "Permanent action"
    Once added to a case, you can't edit or delete observables through TheHive Portal.

<h2>Procedure</h2>

1. [Find and open the case](find-a-case-thehive-portal.md) where you want to add an observable.

    {% include-markdown "includes/filters.md" %}

2. In the case, select the **Observables** tab.

    ![Add observables TheHive Portal](/thehive/images/user-guides/thehive-portal/add-observables-thehive-portal.png)

3. Select :fontawesome-solid-plus:.

4. In the **Add an observable** drawer, enter the following information:

    **- Type \***

    Select the category that best describes what you're reporting.

    **- Value \***

    Enter the specific suspicious information. For example, an email address, a URL, or a phone number.

    To add multiple items at once, place each one on a new line and turn on the **One observable per line** toggle.

5. Select **Add** to add your observable. 

Your observable is visible to all users with access to the case.
