# Delete a Case Comment in TheHive Portal

<!-- md:permission `manageComment` -->

Delete your own comments from cases in TheHive Portal if the information is no longer relevant or if you added it in error.

<h2>Procedure</h2>

1. [Find and open the case](find-a-case-thehive-portal.md) containing the comment you want to delete.

    {% include-markdown "includes/filters.md" %}

2. In the case, select :fontawesome-solid-arrow-left:.

    ![Open comment drawer TheHive Portal](/thehive/images/user-guides/thehive-portal/open-comment-drawer-thehive-portal.png)

3. Select :fontawesome-solid-ellipsis: next to your comment.

4. Select **Delete**.

5. Select **OK**.
