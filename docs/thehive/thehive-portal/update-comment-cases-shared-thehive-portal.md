# Update a Case Comment in TheHive Portal

<!-- md:permission `manageComment` -->

Edit your own case comments after posting them through TheHive Portal to correct information or add additional details.

<h2>Procedure</h2>

1. [Find and open the case](find-a-case-thehive-portal.md) containing the comment you want to update.

    {% include-markdown "includes/filters.md" %}

2. In the case, select :fontawesome-solid-arrow-left:.

    ![Open comment drawer TheHive Portal](/thehive/images/user-guides/thehive-portal/open-comment-drawer-thehive-portal.png)

3. Select :fontawesome-solid-ellipsis: next to your comment.

4. Select **Edit**.

5. Make your changes to the comment.

6. Select **Comment** to save your updated comment.
