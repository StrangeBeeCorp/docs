# Upload an Attachment to a Case from TheHive Portal

<!-- md:permission `manageCase/update` -->

[Attachments](glossary-thehive-portal.md#attachments) allow you to share relevant documents, screenshots, email copies, or other files that provide context or evidence for the security investigation. Remove sensitive information before uploading.

If you need to share specific indicators such as suspicious email addresses, URLs, or phone numbers, see [Add an Observable to a Case from TheHive Portal](add-observable-cases-shared-thehive-portal.md).

!!! note "Permanent action"
    Once added to a case, you can't edit or delete attachments through TheHive Portal.

<h2>Procedure</h2>

1. [Find and open the case](find-a-case-thehive-portal.md) where you want to add an attachment.

    {% include-markdown "includes/filters.md" %}

2. In the case, select the **Attachments** tab.

    ![Add attachments TheHive Portal](/thehive/images/user-guides/thehive-portal/add-attachments-thehive-portal.png)

3. Select :fontawesome-solid-plus:.

4. In the **Add attachment** drawer, drop a file or select it from your computer.

    !!! danger "Sensitive information"
        Ensure the file contains no unintended sensitive information, as it will be visible to all users with access to this case.

5. Select **Confirm**.

Your attachment is visible to all users with access to the case.
