# Overview of Search Methods for Cases

TheHive offers multiple methods for searching [cases](../about-cases.md), each tailored to specific scenarios.

## Search method comparison

| Method | Filters options | Allows bulk actions | Use case |
| -------| ------------------| --------------------| ---------|
| [Search bar](find-a-case.md#method-1-search-bar)| None | No | If you already know the case number or want to conduct simple searches for one or more cases without requiring simultaneous actions.|
| [Similar cases](find-a-case.md#method-2-similar-cases) | Multiple | No | If you want to find one or more cases similar to a known case without needing to perform actions on them simultaneously. |
| [Filters in the Cases view](find-a-case.md#method-3-filters-in-the-cases-view) | Multiple | Yes | If you want to find one or more cases to perform actions on them simultaneously. |
| [Global Search feature](find-a-case.md#method-4-global-search-feature) | Advanced | No | If you need to conduct advanced searches for one or more cases without requiring simultaneous actions. |
