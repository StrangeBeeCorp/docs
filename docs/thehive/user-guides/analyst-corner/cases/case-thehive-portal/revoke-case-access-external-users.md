# Revoke Case Access for External Users

<!-- md:version 5.6 --> <!-- md:permission `manageCaseAccess/external` --> <!-- md:license Platinum -->

Remove external user access to cases when collaboration is complete or no longer required. Revoking access immediately prevents external users from viewing the case and interacting with it through [TheHive Portal](../../../../administration/thehive-portal/about-thehive-portal.md).

<h2>Procedure</h2>

!!! tip "Bulk revocation"
    To revoke access for multiple cases at once, go to the **Cases** view and select :fontawesome-regular-square: next to each case. Then select **Manage external access** above the list.

1. [Locate the case](../search-for-cases/find-a-case.md) where you want to revoke external access.

2. Select the **Manage external access** button.

    ![Manage access case TheHive Portal](../../../../images/user-guides/thehive-portal/manage-access-case-thehive-portal.png)

3. In the **External access management** drawer, select **Internal**.

4. Select **Confirm**.
