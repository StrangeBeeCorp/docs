# Comment on a Case

<!-- md:permission `manageComment` -->

Add comments to [cases](about-cases.md) to document investigation progress, share findings with team members, or provide updates to stakeholders. Comments are visible to all organizations that share the case.

<h2>Procedure</h2>

1. [Find the case](./search-for-cases/find-a-case.md) where you want to add a comment.

2. In the right pane, enter your comment.

    ![Comment on a case](../../../images/user-guides/analyst-corner/cases/comment-case.png)

3. <!-- md:version 5.6 --> <!-- md:license Platinum --> Optional: For cases shared through [TheHive Portal](../../../administration/thehive-portal/about-thehive-portal.md), select the visibility setting from the dropdown list:

    * **Internal** (default): Comment visible only to internal users
    
    * **External**: Comment visible to both internal and external users

    {% include-markdown "includes/visual-indicator-external-comments.md" %}

4. Select **Send** to post the comment.
