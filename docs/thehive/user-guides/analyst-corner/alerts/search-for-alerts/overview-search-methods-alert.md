# Overview of Search Methods for Alerts

TheHive offers multiple methods for searching [alerts](../about-alerts.md), each tailored to specific scenarios.

## Search method comparison

| Method | Filter options | Allows bulk actions | Use case |
| -------| ------------------| --------------------| ---------|
| [Search bar](find-an-alert.md#method-1-search-bar) | None | No | If you need to conduct simple searches for one or more alerts without requiring simultaneous actions. |
| [Similar alerts](find-an-alert.md#method-2-similar-alerts) | Multiple | Yes | If you want to find one or more alerts similar to a known alert and need to perform actions on them simultaneously. |
| [Filters in the Alerts view](find-an-alert.md#method-3-filters-in-the-alerts-view) | Multiple | Yes | If you need to search for one or more alerts to perform actions on them simultaneously. |
| [Global Search feature](find-an-alert.md#method-4-global-search-feature) | Advanced | No | If you need to conduct advanced searches for one or more alerts without requiring simultaneous actions. |
