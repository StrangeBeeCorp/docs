# Share a Knowledge Base Page

<!-- md:version 5.7 -->

Share a [Knowledge Base page](about-knowledge-base.md) link with your teammates through a messaging tool or email.

<h2>Procedure</h2>

1. {% include-markdown "includes/knowledge-base-view-go-to.md" %}

2. Select or hover over the page you want to share in the right pane.

3. Select :fontawesome-solid-link:.

The page link is copied to the clipboard and is ready to share.
